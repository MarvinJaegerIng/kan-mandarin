Kan Mandarin

Im WLAN starten (PC + Handy): Doppelklick auf start-server.bat im Projektordner. Der Server startet, der Browser öffnet sich, und die Adresse fürs Handy wird angezeigt. Einfach das Fenster „Kan Mandarin Server“ offen lassen.

Start the app in two steps:

  1. Open a terminal in this folder and run: npm install
  2. Then run: npm run dev

When you see “Ready” in the terminal, click below to open the app.

Open Kan Mandarin →

If the link doesn’t work, make sure the dev server is running (step 2) and try again.

Auf dem Handy im gleichen WLAN nutzen

Damit du die App auf deinem Android-Handy (oder Tablet) nutzen kannst, muss sie im Netzwerk erreichbar sein.

Variante A – Nur die Standalone-App (ohne npm):

  1. Im Projektordner eine Eingabeaufforderung bzw. Terminal öffnen.
  2. Einen kleinen Webserver starten:
    • Python: python -m http.server 8080
    • Python 3: py -m http.server 8080
    • Node: npx -y serve -p 8080
  3. Die IP-Adresse deines PCs im WLAN herausfinden (z. B. Windows: ipconfig → „IPv4-Adresse“ beim WLAN-Adapter, z. B. 192.168.1.100).
  4. Auf dem Handy den Browser öffnen und eingeben: http://DEINE-IP:8080/app-standalone.html (DEINE-IP ersetzen, z. B. http://192.168.1.100:8080/app-standalone.html).

Variante B – Mit Next.js (npm run dev):

Nach npm run dev auf dem Handy im Browser: http://DEINE-IP:3000 öffnen.

PC und Handy müssen im gleichen WLAN sein. Wenn es nicht geht: Firewall auf dem PC prüfen (Port 8080 bzw. 3000 erlauben).
